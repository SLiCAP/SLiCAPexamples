Introduction#

Combining SLiCAP with Sphinx is a powerful method for writing electronic circuit design documentation websites while doing the design work. Each time, before before compilation, figures, tables, graphs, expressions and equations generated by SLiCAP, are automatically updated and imported in the ReStucturedText (Sphinx) source.

This document briefly describes the way of working. The main document is sphinx/source/index.rst; the path is relative to the SLiCAP project folder. The SLiCAP (Python) source code imported by this document is generated by rstReport.py; in the SLiCAP project folder.

Project file locations#

The project file locations are set and can be altered in the [projectpaths] section of the SLiCAP.ini file in the SLiCAP project folder. Below the listing of [projectpaths] section for this project:

47[projectpaths]
48html = html/
49cir = cir/
50lib = lib/
51csv = csv/
52txt = txt/
53img = img/
54mathml = mathml/
55sphinx = sphinx/
56tex = tex/
57tex_snippets = tex/SLiCAPdata/
58rst_snippets = sphinx/SLiCAPdata/
59html_snippets = sphinx/SLiCAPdata/
60myst_snippets = sphinx/SLiCAPdata/
61md_snippets = sphinx/SLiCAPdata/
62project = /home/anton/DATA/SLiCAP/SLiCAP_github/SLiCAPexamples/Examples/Reports/

Below the relevant paths for making SLiCAP HTML reports with Spihnx. All paths are relative to the project folder.

  • cir: path to circuit netlist files, e.g. netlist files generated with makeCircuit()

  • csv: path to .csv files, e.g. .csv files generated with specs2csv()

  • img: path to image files: .svg, .png, gif, etc., e.g. generated with plot instructions and with makeCircuit()

  • sphinx: path to the Sphinx make files.

    The ReStructuredText source files and the Sphinx configuration file are located in the sphinx/source/ subfolder.

  • rst_snippets: path to RST snippets generated by the rst formatter

  • project: SLiCAP python project folder

Sphinx configuration for SLiCAP#

The Sphinx configuration file conf.py is shown below. You can adapt it to your needs.

 1# -*- coding: utf-8 -*-
 2from time import strftime
 3project = u'SLiCAP Sphinx'
 4html_title = u'SLiCAP Sphinx web reports'
 5html_short_title = u'SLiCAP'
 6copyright = u'2024-2025, Anton Montagne'
 7author = u'Anton Montagne'
 8extensions = ['sphinx.ext.mathjax', 'sphinx_panels', 'sphinxcontrib.bibtex', 'sphinx.ext.autosectionlabel', "sphinx_togglebutton", "sphinx_tabs.tabs"]

10autosectionlabel_maxdepth = None
11bibtex_bibfiles = []
12source_suffix = '.rst'
13master_doc = 'index'
14language = 'English'
15exclude_patterns = ['_build', 'Thumbs.db', '.DS_Store']
16pygments_style = 'sphinx'

18html_last_updated_fmt = strftime("%d/%m/%Y")
19html_domain_indices = True
20html_use_index = True
21html_show_sourcelink = False
22html_show_copyright = True
23math_eqref_format = "({number})"
24numfig = True

Create Sphinx output#

SLiCAP lets you create Sphinx output in two ways:

  • Create images and code listings with SLiCAP functions

    The generation and inclusion of images and code listings be discussed in section SLiCAP output.

  • Use the SLiCAP RST formatter to produce ReStructuredText snippets

    The generation and inclusion of ReStructuredText snippets using the formatter will be discussed in section Formatter methods.
