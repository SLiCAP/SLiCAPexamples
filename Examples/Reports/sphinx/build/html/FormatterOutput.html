Formatter methods#

The ReStructuredText formatter in SLiCAP generates RST snippets that can be imported in RST documents using the .. include:: <file name> role or the variable substitution role |<variable name>|. The RST formatter needs to be initialized. An example of the initialization of this formatter is shown below (see line 11).

 1#!/usr/bin/env python3
 2# -*- coding: utf-8 -*-
 3"""
 4Created on Fri May  2 14:51:28 2025
 5@author: anton
 6"""
 7import SLiCAP as sl
 8import re
 9import os
10sl.initProject("RST formatter") # Initialize the SLiCAP project
11rst = sl.formatter("rst")       # Initialize a ReStructuredText formatter

In this script, variables for inline substitution are saved in the file substitutions.rst in the rst-snippet directory (see ref:Project file locations).

This file is included where ever needed. In this document:

5.. include:: ../SLiCAPdata/substitutions.rst

In the following sections we will discuss the rst formatter methods.

coeffsTransfer()#

coeffsTransfer(Coeffs, label="", caption="", position=0)

This method is used to display the numerator and denominator coefficients of a rational function in the form of a table.

SLiCAP code:

54# Coefficients of the transfer:
55# Define a transfer function:
56H_s = sl.doLaplace(cir).laplace
57# Assign the gain, the normalized numerator coefficients and the 
58# normalized denominator coefficients to the variable 'coeffs'
59coeffs = sl.coeffsTransfer(H_s)
60# Generate a LaTeX snippet of the coefficient table with the 
61# LaTeX formatter 'ltx':
62rst.coeffsTransfer(coeffs, label="tab-coeffs", 
63                   caption="Numerator and denominator coefficients of " +
64                   ":math:`H(s)`, :math:`b_i` and :math:`a_i`, " +
65                   "respectively").save("coeffs")

Let \(H(s)=\) \(\frac{R_{\ell} \left(C_{\mathrm{b}} L s^{2} + 1\right)}{\left(R_{\ell} + R_{\mathrm{s}}\right) \left(\frac{C_{\mathrm{a}} C_{\mathrm{b}} L R_{\ell} R_{\mathrm{s}} s^{3}}{R_{\ell} + R_{\mathrm{s}}} + \frac{s^{2} \left(C_{\mathrm{a}} L R_{\ell} + C_{\mathrm{b}} L R_{\ell} + C_{\mathrm{b}} L R_{\mathrm{s}}\right)}{R_{\ell} + R_{\mathrm{s}}} + \frac{s \left(C_{\mathrm{a}} R_{\ell} R_{\mathrm{s}} + L\right)}{R_{\ell} + R_{\mathrm{s}}} + 1\right)}\)

eqn()#

The formatter method eqn() creates an RST snippet of a displayed and numbered equation.

SLiCAP code:

39# Save the transfer as a LaTeX displayed equation
40rst.eqn("V_out/V_in", transfer, label="eq-H1").save("H1")

RST code:

158The transfer function is shown in :eq:`eq-H1`.
159
160.. include:: ../SLiCAPdata/H1.rst

This renders as:

The transfer function is shown in (1).

(1)#\[\frac{V_{\mathrm{out}}}{V_{\mathrm{in}}} = \frac{R_{\ell} \left(C_{\mathrm{b}} L s^{2} + 1\right)}{\left(R_{\ell} + R_{\mathrm{s}}\right) \left(\frac{C_{\mathrm{a}} C_{\mathrm{b}} L R_{\ell} R_{\mathrm{s}} s^{3}}{R_{\ell} + R_{\mathrm{s}}} + \frac{s^{2} \left(C_{\mathrm{a}} L R_{\ell} + C_{\mathrm{b}} L R_{\ell} + C_{\mathrm{b}} L R_{\mathrm{s}}\right)}{R_{\ell} + R_{\mathrm{s}}} + \frac{s \left(C_{\mathrm{a}} R_{\ell} R_{\mathrm{s}} + L\right)}{R_{\ell} + R_{\mathrm{s}}} + 1\right)}\]

If multiline=True SLiCAP breaks the equation in parts of a sum or a product.
